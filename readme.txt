== Description ==

A simple imagebox plugin for WordPress block editor(gutenberg) with 3 different styles.

* Posts / Pages Imagebox
* No coding skill required
* Fully Customizable
* Fully Responsive
* Multiple Imagebox on same pages


= Future plan = Multiple styles and different hover effects for Imagebox

== Installation ==

How to install the plugin and get it working.


1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
